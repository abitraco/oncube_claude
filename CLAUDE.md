# ONCUBE GLOBAL - Laravel 11 B2B Industrial Equipment Platform

This is a Laravel 11 web application for a B2B industrial equipment and semiconductor component distribution platform with eBay API integration, multi-language support, and an admin dashboard.

## Quick Summary

Laravel 11 B2B platform featuring:

* eBay API integration for product catalog
* Request for Quote (RFQ) system with file uploads
* Featured products admin dashboard
* Multi-language support (en, ko, ja, zh)
* SQLite database
* Docker deployment ready
